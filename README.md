# The Tech Academy Basic C# Projects

This repository contains projects I completed while studying C# and .NET Framework at The Tech Academy. These projects demonstrate my understanding of C#, object-oriented programming, ASP.NET MVC, ASP.NET Core, Entity Framework, and software development concepts.

## Projects

- Package Express – Console application that calculates shipping costs.
- Twenty One Game – Blackjack-style console application.
- Employee Polymorphism – Demonstrates interfaces and polymorphism.
- Operator Overloading – Demonstrates custom operator overloading in C#.
- Car Insurance – ASP.NET MVC web application that generates insurance quotes.
- Server Time App – ASP.NET Core Razor Pages application that displays the current server time.
- Student Code First – Entity Framework Code-First console application that creates a Student database and inserts a student record.
